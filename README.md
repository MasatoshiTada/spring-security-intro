Spring Security Intro
=====================
Spring Securityのサンプルコードです。

# 環境
- JDK 21
- Spring Boot 3.4
- Spring Security 6.4
- Thymeleaf 3.1
- H2 Database

# 実行
[main()メソッド](src/main/java/com/example/springsecurityintro/SpringSecurityIntroApplication.java)から起動

または

```shell
mvn clean spring-boot:run
```
